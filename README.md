# projeto_concorrencia_so
Projeto 01 da Disciplina de SO sobre Sincronização e Concorrência

## Compilando o projeto
```bash
# Compile as dependências
make build

# Compile o código
make

# Caso precise limpar os arquivos
make clean
```

## Executando o projeto
```bash
# Execute o projeto
./main <n_tarefas>

## Exemplo
./main 6
```
